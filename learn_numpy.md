numpy
=========
#### 引入numpy library 
    import numpy as np 
#### np.array()
    製造array 
#### np.arange(0,11,2) 
    製造array  
    array([0, 2, 4, 6, 8, 10])
#### np.zeros(3), np.zeros((5, 5)) 
   array([0., 0., 0.])   
    5*5的零矩陣 
#### np.ones(4) 
    array([1., 1., 1., 1.]) 
#### np.linspace(0, 5, 100) 
    0到5 100個數字分布 
#### np.eye(4) 
    4*4的單位矩陣 
#### np.random.rand(5) 
    隨機5個0~1的浮點數組成的array 
#### np.random.randn(4, 4) 
    4*4 以0為中心點的高斯分布函數 分布在-1~1
